== Description ==

This plugin uses Getaddress.io API to look up UK addresses from postcode. You can sign up for free with 20 API calls per day or you can
choose a plan that starts from £2 per-month.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/uk-address-finder` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 'Settings->Uk Address Finder' to configure the plugin
4. Enter a valid API Key
5. Use the plugin in every page or post with [uk-address-finder]



== Frequently Asked Questions ==

= Do I need to pay in order to get API Key =

No, it is not necessary. GetAddress.io offers a free plan with 20 free API Calls per day. If you want more API Calls, then you must buy a plan that starts from £2 per-month with 50 API Call per day.
